# Glosario Técnico: Semana 1

Aquí tienes las definiciones claras y sencillas de los términos más importantes de esta semana:

---

### Código Fuente
El código escrito en un lenguaje legible para los humanos (como C, Python o JavaScript) antes de ser compilado o interpretado.

### Código Máquina
Instrucciones binarias nativas compuestas exclusivamente de `0`s y `1`s que la Unidad Central de Procesamiento (CPU) ejecuta directamente.

### Compilador
Un software especial que traduce un archivo de código fuente escrito en un lenguaje de alto nivel a código de bajo nivel o binario ejecutable de una sola vez.

### Enlazador (Linker)
Fase final de la compilación que junta distintos módulos de código objeto y librerías externas (como `stdio.h` o `cs50.h`) para formar un único archivo ejecutable.

### Variable
Un contenedor con nombre y tipo definido que apunta a una celda específica en la memoria RAM del sistema para almacenar y manipular datos durante la ejecución.

### Constante
Una variable cuyo valor no puede modificarse una vez definida. En C se declara con la palabra clave `const` (ej: `const float PI = 3.14159;`).

### Desbordamiento de Enteros (Integer Overflow)
Error que ocurre cuando una operación matemática intenta guardar un número mayor al valor máximo que el tipo de datos puede representar en memoria (ej: sumar 1 al valor máximo de un `int`).
